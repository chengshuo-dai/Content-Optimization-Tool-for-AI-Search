# About
An AI Search Content Optimization Tool for xhs content creators.

# Run on website

View on https://cot-for-ai-search.vercel.app/

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the Deepseek API key
3. Run the app:
   `npm run dev`
